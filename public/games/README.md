# public/games/ — 게임 대표 이미지 넣는 곳

각 게임 소개 글의 **대표 이미지(썸네일)** 를 여기에 넣습니다.

## 넣는 방법
1. 게임사 **공식 프레스킷 / 콘텐츠 이용 가이드라인** 에서 이미지를 받습니다.
   - GTA6 → Rockstar Newswire / 프레스: https://www.rockstargames.com/newswire
   - 메챠 카멜레온 → Steam 상점 페이지: https://store.steampowered.com/app/4704690/
2. 파일명을 글에서 참조하는 경로와 **정확히** 맞춰 저장:
   - `gta6.jpg`            (글: 2026-07-11-gta6-confirmed.md → `thumbnail: "/games/gta6.jpg"`)
   - `meccha-chameleon.jpg` (글: 2026-07-11-meccha-chameleon.md)
3. 권장 규격: **가로형 16:9** (예: 1280×720 이상). 세로가 길면 카드에서 위아래가 잘립니다.
4. 파일을 넣고 `npm run build` 하면 카드·시청 페이지에 반영됩니다.
   - 파일이 아직 없으면 자동으로 **그라디언트 플레이스홀더**가 표시됩니다(깨진 이미지 안 뜸).

## 저작권 체크리스트 (인용 요건)
- [x] **출처표기**: 글 frontmatter `imageCredit` 에 출처+링크 기입 → 시청 페이지에 자동 표시
- [x] **부종성**: 본문(자막 소개)이 主, 이미지는 대표 1장만 (從)
- [x] **광고 미표시**: 공식 이미지 쓰는 글은 `ads: false` (상업성↓ → 인용 근거↑)
- [ ] **가이드라인 준수**: 각 게임사 콘텐츠 이용 약관의 허용 범위·조건 확인
- [ ] **takedown 대응**: 게임사 요청 시 즉시 교체/삭제

> ⚠️ 로고·박스아트를 채널 브랜딩(로고처럼)으로 쓰지 말 것. 소개 맥락의 식별용 삽입만.
> ⚠️ 본격 광고(사이트 전체 수익화) 단계에선 AI 창작 이미지로 교체를 권장.
